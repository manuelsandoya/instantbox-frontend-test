# InstantBox — Weather forecast widget challenge

This is the **InstantBox** front-end exercise: implement the widget styling to match the reference image using the rules below.

## Rules

- **Do not** change the markup in `src/App.tsx` (structure, classes, or copy).
- Implement the design by editing **`src/challenge/weather.css` only** (plain CSS; the file includes provided variables and empty selectors).
- There is **no time limit** — you are evaluated on the **final result**, not how long you take.

## Reference

- Reference image: [`public/challenge-target.png`](public/challenge-target.png).
- Run locally: `npm install` then `npm run dev` (dev server uses port **5175**).

## Run checks

- `npm run build`
- `npm run lint`
