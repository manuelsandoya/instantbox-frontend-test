# Weather forecast widget — recruitment exercise

This project mirrors [CSS Daily — Mar 13, 2026 (Weather Forecast Widget)](https://cssdaily.dev/challenge/2026-03-13/).

## Rules

- **Do not** change the markup in `src/App.tsx` (structure, classes, or copy).
- Implement the design by editing **`src/challenge/weather.css` only** (plain CSS; the file includes the official starter variables and empty selectors).
- There is **no time limit** — you are evaluated on the **final result**, not how long you take.

## Reference

- Offline target image: [`public/challenge-target.png`](public/challenge-target.png) (same asset as on CSS Daily).
- Run locally: `npm install` then `npm run dev` (dev server uses port **5175**).

## Run checks

- `npm run build`
- `npm run lint`
